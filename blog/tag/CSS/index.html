Color.js Released

## Announcement It has taken us far too long – there is always something more that _really should_ be added, always some conversion that _really needs_ an authoritive set of data to validate our code against – but we finally decided, time to **actually release** [Color.js](https://colorjs.io/). By which, I guess, we mean removing the _polite figleaf_ that said, “whoops you stumbled on an unreleased thing, please be careful and lower your expectations”. Because people are already using it to make demos and to check other color conversion code against (this includes some browser implementations of [CSS Color 4](https://drafts.csswg.org/css-color-4/) and [CSS Color 5](https://drafts.csswg.org/css-color-5/)).
